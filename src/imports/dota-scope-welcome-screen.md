Главная идея приложения:
Пользователь вставляет ссылку на свой Steam-профиль, например steamcommunity.com/id/thposw/, после чего приложение получает и синхронизирует его Dota 2 статистику. Основной источник данных — OpenDota API. Также предусмотреть визуальное сопоставление данных со стилем статистики, похожим на Dotabuff и OpenDota.

Источники данных:
- получение данных аккаунта по Steam-ссылке профиля
- пример Steam-ссылки: https://steamcommunity.com/id/thposw/
- использовать OpenDota API
- учитывать документацию https://docs.opendota.com/
- учитывать match endpoint по примеру:
  https://api.opendota.com/api/matches/271145478?api_key=29f4b9b4-1bbd-4b6e-84df-af1690f13827
- визуально ориентироваться на статистические паттерны OpenDota и Dotabuff
- статистика должна включать матчи, винрейт, героев, KDA, GPM, XPM, роли, линии, длительность матчей, регионы, частоту побед, прогресс рейтинга и персональные рекомендации
